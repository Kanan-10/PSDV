Installation Guide

Step 1: Download the Installer

Visit the official Altair AI Studio website and download the installer for your operating system.

Step 2: Run the Installer

Locate the downloaded file and double-click it to launch the installation wizard.

Step 3: Select a License and Accept the License Agreement

Select one of the two License Systems. Read and accept the license agreement terms to proceed with the installation.

Step 4: Choose Installation Location

Select the directory where you want to install Altair AI Studio.

Step 5: Complete Installation

Click on "Install" and wait for the process to complete. Once done, launch Altair AI Studio.

Step 6: Setup and First Launch

Open Altair AI Studio, log in with your credentials and start using it.
